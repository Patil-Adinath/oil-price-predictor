# 🛢️ PETROCAST — AI Oil Price Intelligence

PETROCAST is an AI-powered oil price prediction dashboard built 
with React and Claude AI. It analyzes geopolitical tensions, 
market indicators, and global signals to forecast crude oil prices 
across WTI, Brent, and Dubai benchmarks.

## Features
- Real-time price charts (30-day history)
- 7 geopolitical tension sliders (Middle East, Russia-Ukraine, OPEC+, etc.)
- 5 market indicators (Inventories, Rig Count, GDP, Inflation, Nat Gas)
- AI-powered price prediction with confidence score
- Bull/Bear/Neutral sentiment analysis
- Live web search for current market context
- Risk factor breakdown & analyst summary

## Tech Stack
- React (hooks)
- Claude AI API (claude-sonnet-4-20250514)
- SVG custom charts
- Vanilla CSS + Google Fonts
